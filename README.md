Student Performance Prediction using Machine Learning

Project Overview
This project predicts whether a student will Pass or Fail based on:

- Study Hours
- Attendance Percentage
- Previous Marks

A Logistic Regression model is used for classification.


Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
